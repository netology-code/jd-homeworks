# Домашнее задание к занятию «8.5. Spring Data JPA»

## Основное задание

Цель: попрактиковаться в работе с JPA Repositories, переписав приложение для работы с БД [«Слой DAO с Hibernate»](https://github.com/netology-code/jd-homeworks/blob/master/hibernate/task1/README.md).

**Что нужно сделать:**
1.	Перепишите репозиторий для работы с БД на основе методов-запросов:
      -	чтобы в вашем репозитории были основные методы для CRUD операций;
      -	создайте метод, который будет принимать название города (city) и возвращать Entity из базы данных, соответствующие этому city;
      -	создайте метод, который будет принимать возраст (age) и возвращать Entity из базы данных, которые меньше переданного age и отсортированы по возрастанию;
      -	создайте метод, который будет принимать имя и фамилию (name и surname) и возвращать Entity из базы данных, которые соответствуют сочетанию name и surname и являются Optional. 
2. Допишите недостающие методы контроллера в соответствии с новыми методами, появившимися в репозитории.
3. Добавьте миграции для инициализации схемы данных и тестовых данных для проверки работоспособности задания.
4. Добавьте файл docker-compose для запуска СУБД.
5. Написанный код выложите в тот же репозиторий, что и для задачи [«Слой DAO с Hibernate»](https://github.com/netology-code/jd-homeworks/blob/master/hibernate/task1/README.md) на GitHub, только создайте для него другую ветку jpa-repository и прикрепите ссылку на неё в комментарии к домашнему заданию.
